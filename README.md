# BUU — приложение списка дел

## Краткое описание

BUU — это простое мобильное приложение для управления ежедневными задачами. Пользователь может создавать, просматривать и отмечать выполненные дела. Приложение включает базовую навигацию, список элементов, ввод данных и работу с открытым API.

---

## Основные функции

* Добавление новой задачи.
* Просмотр списка задач.
* Отметка выполнения задачи.
* Удаление задачи.
* Навигация между главным экраном и экраном деталей.
* Загрузка вдохновляющей цитаты с публичного API.

---

## Структура экранов

### **1. Главный экран (Список задач)**

* Заголовок: "BUU — Мои задачи".
* Поле ввода новой задачи.
* Кнопка "Добавить".
* Список задач (каждая строка: текст + чекбокс + кнопка удаления).
* Кнопка перехода на экран деталей.

### **2. Экран деталей / информации**

* Заголовок: "Детали / Информация".
* Текст вдохновляющей цитаты, загружаемой по API.
* Кнопка "Назад".

## Навигация

* **Главный экран → Экран деталей** — кнопка "Перейти к деталям".
* **Экран деталей → Главный экран** — кнопка "Назад".

---

## Сценарии использования

### **Сценарий 1: Добавление новой задачи**

1. Пользователь вводит текст в поле.
2. Нажимает кнопку "Добавить".
3. Новая задача появляется в списке.

### **Сценарий 2: Отметка выполнения задачи**

1. Пользователь нажимает на чекбокс рядом с задачей.
2. Статус задачи обновляется.

### **Сценарий 3: Удаление задачи**

1. Пользователь нажимает кнопку удаления (X).
2. Задача удаляется из списка.

### **Сценарий 4: Загрузка вдохновляющей цитаты**

1. Пользователь открывает экран деталей.
2. Приложение отправляет запрос к публичному API.
3. На экране отображается цитата.

### **Сценарий 5: Навигация между экранами**

* Кнопки позволяют переходить туда и обратно.

## Публичный API

Для загрузки данных используется публичный API вдохновляющих цитат:

**Inspiration API** — мотивационные и вдохновляющие цитаты

* Auth: нет
* HTTPS: да
* CORS: да
  Документация: [https://inspiration.goprogram.ai/docs/](https://inspiration.goprogram.ai/docs/)
